# Sincronizar con Obsidian (iPhone) — vía Git

Instrucciones rápidas para abrir este repositorio en Obsidian Mobile usando la app Working Copy (recomendado):

1. Instalar Working Copy (iOS) y Obsidian (iOS).
2. En Working Copy: "Clone repository" → pegar: https://github.com/LEAN-EnLAN/scribe.git
3. Esperar a que termine el clone.
4. Abrir Obsidian Mobile → "Open vault" → "Open folder in Files" → seleccionar la carpeta clonada dentro de Working Copy.
5. Para sincronizar desde el iPhone: usar Working Copy para commit & push; luego en desktop: pull.

Opcional (desktop): instalar plugin "Obsidian Git" para pushes/commits automáticos desde tu máquina.

Notas:
- También podés usar Obsidian Sync (de pago) si preferís sincronización nativa.
- Usar ramas para experimentos y PRs si trabajás con otras personas.
